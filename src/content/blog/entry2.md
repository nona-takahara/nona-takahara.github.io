### 追加した構文を使いたいだけ
__いつも__ テスト用のWebサーバの立て方を忘れるので、備忘録も兼ねて。
ローカル環境ではいつもPythonの`http.server`モジュールを使ってテストしてます。
```
cd build
python -m http.server 8000
```
こうやって備忘録としてちゃんと書くと、意外と覚えてしまうような気がしてきます。
